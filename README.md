## üöÄ Quick Start

### Prerequisites
- Node.js 18+
- DeepSeek API key
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/ECommerce-chat-agent.git
cd ECommerce-chat-agent
```
2. **Backend Setup**
```bash
cd backend
npm install
cp .env.example .env
# Add your DeepSeek API key to .env
npm run dev
```

3. **Frontend Setup**
```bash
cd frontend
npm install
npm start
```
